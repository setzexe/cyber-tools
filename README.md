# cyber-tools

A collection of cybersecurity tools created with the intent of learning, experimentation, and practical usage.

This repsitory is part of a broader personal cybersecurity learning effort, which will continue to evolve as more topics and tools are learned.

## Tool Purpose

This repository focuses on tools that can tackle tasks or issues brought up in red / blue team situations. The intent and scope of the tools remain clear, with an emphasis on learning and correctness over speed. As I learn more and more about a certain topic, the tools will be improved over time via versioning.
